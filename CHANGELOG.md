## [1.0.0] — 2024-01-15

### Добавлено
- SKILL.md — главный файл скилла с двумя уровнями анализа
- Level 1: Note Intelligence (анализ одной заметки / небольшой пачки)
- Level 2: Vault Intelligence с 5 фазами анализа
- AI Creator Lens: Build It / Teach It / Compound It тесты
- references/ai-creator-taxonomy.md — таксономия AI-ниш и архетипов
- references/vault-structures.md — структуры vault + MOC-шаблоны
- references/opportunity-patterns.md — 15 паттернов поиска возможностей
- 5 шаблонов Obsidian: permanent-note, literature-note, moc, competency-map, opportunity-log
- 3 примера: single-note-analysis, vault-intelligence-sample, before-after-notes
- export-vault.py — Python скрипт для экспорта vault
- docs/effective-usage.md — руководство по эффективному использованию
- docs/how-it-works.md — техническое объяснение логики скилла
- docs/zettelkasten-primer.md — введение в Zettelkasten
- docs/faq.md — частые вопросы
